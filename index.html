It's a Quick GCLOUD Command Line Memo.

General

log out from cloud shell
exit

print version
gcloud version

lists credentialed account
gcloud auth list

list the roles
gcloud iam roles list

list project's service accounts
gcloud iam service-accounts list

organization / list organizations
gcloud organizations list

list projects
gcloud projects list

list current project(Cloud SDK property)
gcloud config list project

set a project(Cloud SDK property)
gcloud config set project PROJECT_ID

list GCE regions
gcloud compute regions list

list current region(Cloud SDK property)
gcloud config list compute/region

set a region(Cloud SDK property)
gcloud config set compute/region REGION

list GCE zones
gcloud compute zones list

list current zone(Cloud SDK property)
gcloud config list compute/zone

set a zone(Cloud SDK property)
gcloud config set compute/zone ZONE

Compute

list compute engine instances
gcloud compute instances list

describe compute engine instances
gcloud compute instances describe INSTANCE_NAME

start a compute engine instance
gcloud compute instances start INSTANCE_NAME

stop a compute engine instance
gcloud compute instances stop INSTANCE_NAME

SSH into a compute engine instance
gcloud compute ssh INSTANCE_NAME

copy files to and from compute engine instance via scp
gcloud compute scp [INSTANCE_NAME:]SOURCE_FILE [INSTANCE_NAME:]DESTINATION_NAME

list compute engine firewall rules
gcloud compute firewall-rules list

list compute engine forwarding rules
gcloud compute forwarding-rules list

list compute engine machine types
gcloud compute machine-types list

list compute engine images
gcloud compute images list

list compute engine templates
gcloud compute instance-templates list

list compute engine instance groups
gcloud compute instance-groups list

list instances in the instance group
gcloud compute instance-groups list-instances NAME

Container

fetch credential
gcloud container clusters get-credentials CLUSTER_NAME

list container clusters
gcloud container clusters list

describe container clusters
gcloud container clusters describe CLUSTER_NAME

list GKE images
gcloud container images list

list node pools for container cluster
gcloud container node-pools list

Kubectl

list services
kubectl get services

list deployments
kubectl get deployments

list replicasets
kubectl get replicasets

list hpa
kubectl get hpa

list pods
kubectl get pods

list nodes
kubectl get nodes

App Engine

deploy the code to App Engine
gcloud app deploy

open the app in web browser
gcloud app browse

list versions of App Engine
gcloud app versions list

list services of App Engine
gcloud app services list

list instances of App Engine
gcloud app instances list
